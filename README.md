# 配置

## 写入对应配置文件

```bash
yarn add dotenv-cli --dev
```

## 配置 less

```bash
yarn add less-loader less --dev
```

## 打包进度条

webpack 插件

```bash
yarn add progress-bar-webpack-plugin --dev
```
